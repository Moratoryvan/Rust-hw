# README 

## 如何在本地测试？

命令行输入：
```
$ git clone https://github.com/Moratoryvan/Rust-hw.git
$ cd Rust-hw/hw3
$ cargo test 
```

## 文件路径

```
hw3
├─ src
│  ├─ hash_map.rs           // 实现 hash 表
│  ├─ main.rs               // 测试
│  ├─ MyRc.rs               // 实现 MyRc 智能指针
│  └─ stack.rs              // 实现栈
├─ Cargo.lock
├─ Cargo.toml
└─ README.md
```
