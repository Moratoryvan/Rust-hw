# README 

## 如何在本地测试？

命令行输入：
```
$ git clone https://github.com/Moratoryvan/Rust-hw.git
$ cd Rust-hw/hw4
$ cargo test 
```

## 文件路径

```
hw4
├─ src
│  ├─ executor.rs           
│  ├─ main.rs               // 测试
│  ├─ waker.rs               
├─ Cargo.lock
├─ Cargo.toml
└─ README.md
```
