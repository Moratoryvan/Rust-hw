# README 

## 如何在本地测试？

命令行输入：
```
$ git clone https://github.com/Moratoryvan/Rust-hw.git
$ cd Rust-hw/hw2
$ cargo test 
```

## 文件路径

```
hw2
├─ src
│  ├─ buffer.rs           // 实现 Buffer
│  ├─ compare.rs          // 实现字符串比较
│  └─ main.rs             // 实现闭包迭代器结合，并对三个练习做测试
├─ Cargo.lock
├─ Cargo.toml
└─ README.md
```
